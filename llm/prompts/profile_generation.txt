Your task is Semantic Profile Generation for SASE.

===============================================================================
BACKGROUND
===============================================================================

SASE is a semantic-guided symbolic execution framework.

The purpose of this stage is to build a semantic profile for a target function.

This is ONLY the first stage of Semantic Objective Construction.

Do NOT infer behavioral progressions.

Do NOT rank objectives.

Do NOT generate semantic traces.

Those tasks belong to later stages.

===============================================================================
GOAL
===============================================================================

Your goal is to summarize the semantic role of the target function under the
current execution context.

The semantic profile should provide a high-level abstraction of the function's
behavior instead of describing implementation details.

The semantic profile will later be used to retrieve domain knowledge and infer
semantic exploration objectives.

===============================================================================
INPUTS
===============================================================================

You are given:

1. Target function source code

2. Relevant source code on the current call stack

3. Semantic tags generated during Semantic Tag Creation

The semantic tags describe semantic events that may occur inside the function.

They should be treated as semantic hints instead of implementation details.

===============================================================================
WHAT IS A SEMANTIC PROFILE?
===============================================================================

A semantic profile summarizes:

- why the function exists

- what semantic responsibility it has

- what semantic behaviors it performs

- how it contributes to the overall program

It should describe the function at the behavioral level rather than the
instruction level.
